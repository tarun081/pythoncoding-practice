# Coding Practice

A collection of coding problem solutions implemented in Python.

The goal of this repository is to improve problem-solving skills, strengthen python , Data Structures & Algorithms concepts, and maintain a consistent coding practice routine.

## Goals

- Solve and document coding problems regularly.
- Build strong problem-solving skills.
- Prepare for coding interviews and online assessments.
- Learn multiple approaches for solving the same problem.
- Complete the first 100 coding problems and continue expanding the repository.

## Setup Guide

### Prerequisites

- Python 3.10+ (recommended)
- Git

### Verify Installation

```bash
python --version
git --version
```

### Clone Repository

```bash
git clone <repository-url>
cd coding-practice
```

### Create Virtual Environment (Optional)

#### Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Run a Solution

```bash
python codingproblems/problem_001.py
```

## Coding Standards

- Keep solutions simple and readable.
- Add comments where necessary.
- Prefer optimal solutions after implementing brute-force approaches.
- Follow Python best practices and PEP 8 guidelines.
